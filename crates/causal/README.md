# causal (Antecedent facade)

Unified `CausalAnalysis` facade: static ATE (DAG or PAG/generalized adjustment),
temporal effect, distribution / path-specific, mediation, conditional effect,
counterfactual and attribution queries — identify → estimate (→ refute) with
logical/physical planning. Public product / Python package: **Antecedent**
(`import antecedent`; PyO3 crate `causal-py`).
