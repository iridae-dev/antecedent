# antecedent

Day-1 Rust facade for **Antecedent**: identify → estimate → refute with logical/physical
planning (static and temporal queries).

```toml
antecedent = "0.1"
```

```rust
use antecedent::prelude::*;
```

Supporting libraries remain `causal-*` crates on crates.io and are **public
dependencies** of this facade (part of the 0.1 semver surface). The Python
extension (`causal-py`) is not published to crates.io.

See the workspace root README and `CHANGELOG.md`. `0.1.x` may still introduce
breaking changes.
